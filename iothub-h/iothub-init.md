# IoTHub_Init()

IoTHubClient_Init Initializes the IoTHub Client System.

## Syntax

\#include "[azure-iot-sdk-c/iothub_client/inc/iothub.h](../iothub-h.md)"  
```C
int IoTHub_Init(void
);
```

## Return Value
int zero upon success, any other value upon failure.
