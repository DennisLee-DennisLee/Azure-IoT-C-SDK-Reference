# Map_Destroy()

Release all resources associated with the map.

## Syntax

\#include "[azure-iot-sdk-c/c-utility/inc/azure_c_shared_utility/map.h](../map-h.md)"  
```C
void Map_Destroy(
  MAP_HANDLE  handle
);
```

## Parameters
* `handle` The handle to an existing map.
